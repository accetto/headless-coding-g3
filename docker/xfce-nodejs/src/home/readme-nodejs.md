# Readme `Node.js`

## Updating `npm`

You can update `npm` by:

```shell
npm install -g npm
```

You can check the current versions by:

```shell
node -v
npm -v
npx -v
```

## What next

There is a sample application `nodejs-test-app` in `/srv/samples`. You can try it following the [Node.js Guides](https://nodejs.org/en/docs/guides/).
